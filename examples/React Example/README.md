**Getting Started:**

This application uses icons from Font Awesome. To make sure these show up / don't throw an error:

In sdk-internal:

`npm install --save font-awesome
`

In `src/Auth.js` replace the `API_KEY` and `API_SECRET` variable values with your application's credentials.

**To Run:**

cd into reactexample

cd into src

Use the following commands:

`npm install`

`npm start`
